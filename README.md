# Futuro do Trabalho — Rede Profissional

## Resumo do projeto
Aplicação front-end (HTML, CSS e JavaScript) que simula uma rede profissional voltada ao futuro do trabalho.
Apresenta 10 perfis fictícios (array), cards dinâmicos, modal com slideshow de detalhes e um Quiz melhorado com explicações e pontuação.

## Conteúdo do repositório
- `index.html` — página principal
- `styles.css` — estilos (Flexbox, responsividade, animações)
- `script.js` — lógica: array de perfis, criação de cards, modal/slideshow e quiz
- `README.md` — este arquivo

## Como rodar
1. Clonar o repositório:
```
git clone <https://github.com/Gguicruzz/futuro_trabalho_full/>
cd futuro_do_trabalho
```
2. Abrir `index.html` no navegador ou rodar servidor local:
```
python -m http.server 3000
# abrir http://localhost:3000
```
